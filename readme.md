# Hotel Reservations

This is a web application used to make hotel reservations.

- Built in go 1.25
- Uses [scs](https://github.com/alexedwards/scs/v2)
- Uses [chi](https://github.com/go-chi/chi)
- Uses [nosurf](https://github.com/justinas/nosurf)
- CI/CD Implemented
  - CI - Run tests
  - Deploy - AWS EC2
